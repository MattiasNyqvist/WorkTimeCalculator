# Work Time Calculator

A web application built with Blazor Web App (.NET 10) that calculates when you can leave work based on your working hours, start time, and lunch break.

## Features

- Input working hours in decimal format (e.g. 7.7 or 7,7)
- Input start time
- Input lunch break in minutes
- Calculates and displays your end time

## Tech Stack

- Blazor Web App (.NET 10)
- InteractiveServer render mode
- Bootstrap 5
- C#

## Getting Started

### Prerequisites

- .NET 10 SDK

### Run locally

```bash
cd src/WorkTimeCalculator
dotnet watch
```

Open your browser and navigate to `http://localhost:5223`
